Aprendendo Flexbox

O que é Flexbox?

Flexbox é um modelo de layout unidimensional do CSS usado para alinhar os itens no container.

Basta aplicar display: flex ao elemento pai e os filhos automaticamente se organizam em linhas e colunas.

Por que usar?

Facilita na aprendizagem e desenvolvimento de colunas e layouts.
